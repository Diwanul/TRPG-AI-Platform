### 1. IDE 概念的重要性
```
平凡的游戏工具：游戏界面 + 菜单
TRPG IDE：游戏界面 + 记忆系统 + 资源库 + AI
```

### 2. Prompt Engineering 的力量
```
传统方法：代码 → 游戏逻辑（维护困难）
AI 方法：Prompt → 无限可能（易于扩展）
```

### 3. 外置记忆的价值
```
临时游戏：当前场景
游戏 IDE：完整历史 + 角色档案 + 决策树
```

### 4. 零成本架构
```
传统 Web 应用：$10-100+/月
TRPG IDE：$0（用户成本可控）
```
